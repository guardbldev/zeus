# Zeus - **A Roblox Terminal Plugin**

Zeus is an extensible command-line framework for Roblox Studio that provides automation, static analysis, security auditing, performance profiling, and project validation through a powershell-like terminal user interface. From code generation and refactoring to security audits and build checks, this plugin brings development workflows into Roblox Studio.

### **Command System**

- PowerShell-style UI command syntax
- Flags, arguments, quoted strings
- Command history & replay
- Context-aware execution (Selection, active script, environment)
- Tab autocomplete & fuzzy matching

### **Plugin UI Features**

- VS Code–style embedded terminal
- Streaming output & progress bars
- Color-coded logs (info / warning / error)
- Clickable stack traces → jump to scripts
- Persistent command history per project
